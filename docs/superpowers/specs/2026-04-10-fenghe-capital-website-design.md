# 峰壑资本俱乐部官网设计规范 (Fenghe Capital Website Design Spec)

## 1. 核心定位与视觉基调
- **定位**：顶级数字门户、价值引擎与全球声望门户。
- **视觉基调 (Visual Thesis)**：极暗黑金（Dark Mode Only）、高对比度、无卡片（Cardless）边缘极简设计。大量运用摄影级大图与留白，剔除常规 SaaS 的繁杂框线，辅以纯粹黑色深渊与暗金色光影交互。
- **交互基调 (Interaction Thesis)**：电影级视差滚动与浮现。页面过场摒弃生硬的白底黑字加载，运用 `framer-motion` 实现长卷翻阅的极致平滑度和字面悬浮感（如：呼吸输入框、文字逐行渲染、横向铺展等）。

## 2. 路由与全局导航结构
遵循 PDF 官方命名标准，核心路由设立以下七大板块：
- `/`：首页
- `/philosophy`：峰壑哲学
- `/os`：资本操作系统
- `/galaxy`：峰壑星系
- `/alliance`：共筑峰峦
- `/vision`：峰壑视野
- `/contact`：共筑峰峦入口（联系我们 / 申请表单）

## 3. 详细页面结构
### 3.1 首页 (/)
- **引力核**：100vh 云海全画幅视频，极简压暗处理，展示核心标语“资本遇见雄心，我们共筑峰峦”。
- **峰壑宣言**：纯黑悬浮渲染交互，“真正的资本是加速器，并非收割器”。
- **三大价值**：全景图文叠层纵向铺开，悬停亮起金色等高线。
- **峰壑之问**：桌面设计视角微视频片段，引导转化按钮。

### 3.2 峰壑哲学 (/philosophy)
- **视觉**：“平原思维”与“峰峦思维”极简黑白图形对比。
- **价值观排版**：大字幕高宽容度排版（第一性原理、全球化智慧），突出纯净留白。
- **创始人致信**：全屏下探纵深，拟真手迹滑动展现仪式感。

### 3.3 资本操作系统 (/os)
- **核心交互**：自上而下的鼠标滚轮转换为横向位移（Horizontal Scroll），带领用户穿越“融、投、管、退”四大信息岛屿舱体。
- **强风控数据面板**：半掩深色磨砂面板，刻冰冷金色数字面板，突出“上市百分百”、“亏损兑付”等硬核承诺实力标签。

### 3.4 峰壑星系 (/galaxy)
- **极核微距生态图**：极简粒子/光影全国数字地图呈现。
- **区域探照面板**：悬停地图闪点光斑时，丝滑弹出无缝底侧面板信息展示各地分支力量。
- **生态 Logo 阵列**：深色压暗的单色底纹外部合作伙伴 Logo 横向巡游展出。

### 3.5 共筑峰峦 (/alliance)
- **大图阵式栏**：不使用网格小卡片。对企业方、生态合伙人、专属机构会员用全幅画面与版切呈现差异化权利和门槛。
- **数字力量**：用巨大醒目的业绩数字撑起成功展示；只提炼顶级权限重点词。

### 3.6 峰壑视野 (/vision)
- **顶级读本化视界**：极具反差的无边框长卷册。内容单列瀑布流展开，主要展示《登峰》与《第一性原理》系列内容。
- **沉浸阅读体验**：单列高质封面和吸睛大标题，点击后在当前页无框模糊渐隐过渡到正文细读。

### 3.7 共筑峰峦入口 (/contact)
- **高阶联系页**：采用非对称分割双栏空间 (40/60 比例 Split View)。
- **留白视觉侧（40%）**：展示总部物理位置图表信息、高管级别对接邮箱。
- **极简交互侧（60%）**：无框下划线表单提问“希望以什么身份联结？”。智能切换：根据身份选择（项目 BP 投递 / 加盟申请 / 机构共建）动态渐出相应的专属内容投递通道模块，顺延支持高阶附件极简拖拽。

## 4. 技术规范说明
- **框架**：Next.js App Router (React)
- **样式**：Tailwind CSS / shadcn/ui 无边框组件级定制
- **动画引擎**：Framer Motion 负责高级视差、排版渲染、页面横纵切换及细节浮现交互。
